# Roadmap

Future improvements intentionally left out of the first implementation:

- image cache pruning
- automated template rebuilds
- CI linting for shell scripts
- Packer-based builds
- OpenTofu or provider-based image upload workflow
- signed release notes for known-good template versions

Keep the first version simple: one repeatable Proxmox template flow, generic image profiles, and a clean handoff to the infrastructure repository.
